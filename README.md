# image-tile-server
a simple image tile server that uses mapbox-gl-native

# install
```console
git clone git@github.com:hfu/image-tile-server
git clone git@github.com:hfu/onyx-tapioca
cd image-tile-server
npm install
```

# use
```console
node index.js
```
